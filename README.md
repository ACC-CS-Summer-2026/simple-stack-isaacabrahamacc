# Assignment 4: Simple Int Stack

## Description
A C++ implementation of a bounded, static array-based integer Stack Abstract Data Type (ADT). This program features robust error-checking mechanisms to handle stack overflow and underflow conditions safely using pointer interfaces.

## Features
- Bounded integer array storage.
- Safe data retrieval via integer pointer validation states.
- Dynamic tracking evaluation loop scaling dynamically based on macro definitions.
